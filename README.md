# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/admindee.git
```

Go into project:

```bash
cd admindee
```

Install PHP dependencies:

```bash
composer install
```

Install Node dependencies:

```bash
npm install
```

Copy environment:

```bash
cp .env.example .env
```

Generate app key:

```bash
php artisan key:generate
```

Run migrations:

```bash
php artisan migrate
```

Start development server:

```bash
php artisan serve
```

Run Vite:

```bash
npm run dev
```
